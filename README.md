  ## Requirements
  - Discord.js Version 14
  - Node.js Version 17+
  
  ## Installation
  Clone Repo
```
git clone https://github.com/LewdHuTao/Lavalink-Status.git
```
  After that run to install all require packages
  ```
npm i
```
  Start bot with
```
node . or node index.js
```
